## 关键 code/llm 内容

### 6250 LLM prompt
```
system: 你是反馈内容整理助手。把用户输入转写成标准中文反馈,并提取2-3个用于查表的关键词。
user: 用户输入: {{#6002.query_text#}}
请输出 JSON: {"feedback_zh":"中文转写","keyword":"关键词1,关键词2"}
```
outputs: feedback_zh(string), keyword(string)。注意 LLM 输出 JSON 需 code 解析,或用 Dify structured output。

### 6240-parse code
```python
def main(http_body: str) -> dict:
    import json
    try:
        data = json.loads(http_body)
    except:
        return {"hit_record_id": "", "row_summary": ""}
    hits = data.get("hits") or []
    if not hits:
        return {"hit_record_id": "", "row_summary": ""}
    h = hits[0]
    return {
        "hit_record_id": h.get("record_id", ""),
        "row_summary": h.get("op_desc", "") or h.get("summary", "")
    }
```
outputs: hit_record_id(string), row_summary(string)

### 6244-timer code
```python
def main(llm_text: str, feedback_zh: str) -> dict:
    # feedback_zh 截断防标记过长
    fz = (feedback_zh or "")[:200]
    marker = f"<!--SYS:TIMER|action=arm|state=await_confirm_new|feedback_zh={fz}-->"
    return {"answer_text": (llm_text or "") + "\n" + marker}
```
outputs: answer_text(string)
